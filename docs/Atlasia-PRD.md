# PRD — Site de Geografia (Catálogo de Países)

## 🧩 Contexto

Projeto pessoal com objetivo educacional, aberto ao público, focado em consulta de informações sobre países (bandeiras, capitais, etc.), com possibilidade futura de evolução para quizzes.

## 🎯 Objetivo (com métricas)

- Disponibilizar catálogo completo de países do mundo
- Tempo médio na página > 2 min
- Taxa de retorno > 30%

## 👤 Persona

- Estudantes
- Curiosos por geografia
- Usuários que querem memorizar países e capitais

## 😖 Problema

Falta de uma ferramenta simples, bonita e direta para consulta rápida e memorização de informações geográficas.

## 💡 Solução

Criar um site minimalista e responsivo com listagem completa de países, busca, filtros e páginas individuais com dados relevantes.

## 🚀 MVP (apenas essencial)

- Lista de países
- Busca por nome
- Página individual de país
- Informações básicas
- Layout responsivo

## 📦 Funcionalidades

- Listagem completa
- Busca por país
- Filtro por continente
- Página de país com:
  - Bandeira
  - Nome
  - Capital
  - Continente
  - Idioma
  - Moeda
  - População

- Design minimalista

## 🔁 Fluxo do Usuário

Usuário entra → vê lista → busca ou navega → clica em país → visualiza detalhes

## ✅ Critérios de Aceitação

- Lista carrega corretamente
- Busca retorna resultados relevantes
- Página do país exibe dados corretamente
- Funciona em mobile

## 📊 Métricas

- Tempo médio na página
- Número de páginas visitadas

## ⚠️ Riscos e Assunções

- Fonte de dados pode variar
- Escopo pode crescer rápido

## 🔮 Próximos Passos

- Adicionar quizzes
- Sistema de progresso
- Multi-idioma
